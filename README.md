# Sample app using react-native-biometrics

## Get started

```zsh
  git clone https://github.com/YutaSugimura/react-native-biometrics-sample-app.git
  cd react-native-biometrics-sample-app
  yarn install
  npx pod-intall

  yarn start

  # iOS
  yarn ios

  # Android
  yarn android
```
